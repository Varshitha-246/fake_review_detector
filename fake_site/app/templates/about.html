About Fake Review Detector

The Fake Review Detector is an AI-powered web application designed to analyze product reviews and determine whether they are genuine or fake. In recent years, online shopping platforms have faced an increasing challenge due to fake or misleading reviews that influence user purchasing decisions. This system helps users identify unreliable or bot-generated reviews using modern deep learning techniques.

How It Works

The tool uses a fine-tuned DistilBERT transformer model, trained on a labeled dataset containing both deceptive and truthful online reviews. The model analyzes linguistic patterns, sentiment tone, writing repetitiveness, exaggeration cues, and contextual phrasing to classify a given review as:

✔ GENUINE — Natural human expression with balanced opinions
✔ FAKE — Artificial, exaggerated, repetitive, or bot-like text

Dataset Used

The dataset used includes thousands of annotated reviews with balanced labels for Fake and Genuine content. It was preprocessed with tokenization, normalization, and input encoding before training the model.

Model & Techniques

- DistilBERT Transformer model
- Tokenization using HuggingFace libraries
- Fine-tuned classification layers
- Evaluation via F1-Score, Precision, Recall, and Confusion Matrix

Technologies Used

✔ Python • Django • HuggingFace Transformers • PyTorch • HTML5/CSS3

This project aims to provide a meaningful contribution toward reducing misinformation in online shopping ecosystems by assisting customers in identifying reviews that may not reflect real product experiences.
